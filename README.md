# Hermes Crypto Agent — Autonomous Trading Bot Skill

> **Turn your Hermes Agent into a profitable crypto trading machine.**
> 
> Automated grid trading, DCA strategies, smart money tracking, and multi-exchange support — all running 24/7 on OKX with zero manual intervention.

## 💰 What This Skill Does

- **Grid Bot Automation** — Create, manage, and optimize OKX grid bots with AI-recommended parameters
- **DCA (Martingale) Strategies** — Smart dollar-cost averaging with dynamic safety order sizing
- **Smart Money Tracking** — Follow whale wallets and top traders automatically
- **Multi-Timeframe Analysis** — RSI, volume spike, and trend detection for entry/exit timing
- **Risk Management** — Auto stop-loss, take-profit, and position sizing based on account balance
- **24/7 Monitoring** — Cron job integration for continuous market scanning

## 🚀 Quick Start

```bash
# Install the skill
hermes skills install a918124259a/hermes-crypto-agent

# Initialize with your OKX API credentials
okx config init

# Start a BTC grid bot with AI-recommended parameters
hermes chat "Create a BTC-USDT grid bot on OKX between $60k-$70k with 20 grids, 500 USDT investment, 3% TP"
```
